# Expense Splitter Application

A full-stack expense splitting application where users can create groups, add expenses, and automatically calculate who owes whom.

## Technology Stack

### Backend
- Node.js with Express
- TypeScript
- In-memory storage
- express-validator for validation

### Frontend
- React 18
- TypeScript
- Vite
- shadcn/ui components
- Tailwind CSS
- React Router

## Project Structure

```
expense-splitter-application/
├── backend/          # Express API server
└── frontend/         # React application
```

## Getting Started

Detailed setup instructions will be added as the project develops.

## Features

- Group management (create, view, update, delete)
- Member management within groups
- Expense tracking with multiple split types (equal, percentage, exact)
- Balance calculation
- Optimized settlement suggestions
